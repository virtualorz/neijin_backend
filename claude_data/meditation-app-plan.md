### 全球市場
- 2025 年全球冥想 App 市場規模約 **27 億美元**，年均複合成長率（CAGR）約 **25%**
- 預測 2033 年達 158 億美元
- 訂閱制佔市場收入約 58%

### 台灣市場
- 台灣「心經濟」崛起，全球療癒市場 2025 年逼近 NT$136 兆，台灣同步受惠
- 全球前三大競品（Calm、Headspace、Insight Timer）合計佔市場 77%，**三者均無繁體中文版本**
- 台灣目前無本土繁中冥想 App 品牌，市場空白明顯

## 五、功能規劃

### 免費版（每天都打開的黏著功能）

#### 1. 每日 5 分鐘晨間冥想
- AI 語音引導冥想，固定一套劇本
- ATEN 優聲學生成靜態音頻，所有免費用戶共用同一檔案
- 目的：讓用戶習慣品牌聲音、建立每日使用習慣

#### 2. 呼吸練習計時器
- 視覺化圓圈動畫：放大（吸氣）→ 縮小（吐氣）
- 支援呼吸模式：4-7-8、Box Breathing（4-4-4-4）、2-4 基礎版
- 純前端實作，零 TTS 成本

#### 3. 白噪音播放器（基礎版）
- 3–5 種聲音：雨聲、咖啡廳環境音、海浪、森林鳥叫
- 來源：Pixabay Audio（CC0 授權）
- 靜態音頻，CDN 無限播放

#### 4. 番茄鐘計時器
- 搭配白噪音使用：25 分鐘工作 + 5 分鐘休息
- 工作時播放白噪音，休息時自動切換輕鬆音樂或引導呼吸
- 純前端計時邏輯，零成本

---

### 付費版（訂閱解鎖）

#### 1. 場景化引導冥想（核心付費內容）
用 ATEN 批量生成多套台灣腔音頻，按情境分類：

| 情境 | 時長 |
|------|------|
| 睡前深度放鬆 | 15 分鐘 |
| 上班前清醒冥想 | 8 分鐘 |
| 通勤 / 捷運冥想 | 10 分鐘 |
| 工作壓力緊急釋放 | 5 分鐘 |
| 午休快速恢復 | 10 分鐘 |
| 焦慮時刻緊急引導 | 5 分鐘 |
| 月前症候群情緒疏導 | 12 分鐘 |

目標：10 個情境 × 平均 10 分鐘 = **約 100 分鐘付費內容庫**

#### 2. 睡眠故事
- 類似 Calm 的 Sleep Stories
- ATEN 語音朗讀 20–30 分鐘舒緩故事，配 Pixabay 輕音樂背景
- 初期目標：3–5 個故事主題

#### 3. 進階白噪音（15–20 種）
免費版 3–5 種升級為：

> 書頁翻動聲、鍵盤打字聲、圖書館環境音、篝火聲、日式溫泉聲、
> 山林溪流聲、夜晚蟲鳴、輕雷雨聲、茶屋環境音、輕爵士咖啡廳

#### 4. 情緒主題課程（系列冥想）
- 按週期設計，形成用戶習慣迴路
- 範例：「7 天減壓課程」、「14 天睡眠改善」、「21 天正念入門」
- 每集 8–10 分鐘，搭配每日提醒通知

#### 5. 情緒日誌 + AI 推薦
- 每日情緒打卡（1–5 分）
- 根據情緒分數推薦對應冥想（低分 → 放鬆冥想，高壓 → 緊急釋放）
- 情緒折線圖追蹤，連續打卡天數（streak）

## 七、MVP 開發路線圖

### Phase 1：Week 1–2（核心 MVP）
- 靜態 Web App：5 首引導冥想 + 3 種白噪音 + 呼吸計時器
- 用 ATEN 免費方案生成初版音頻
- 技術：Next.js + Supabase + Web Audio API
- 重點：**驗證用戶是否接受 AI 語音的冥想體驗**

### Phase 2：Week 3–5（訂閱牆 + 情緒日誌）
- 免費 → 付費差異化（場景化冥想課程庫解鎖）
- 訂閱金流：綠界 ECPay（定期扣款）
- 情緒打卡（1–5 分）+ 連續天數 streak 顯示
- Claude API 推薦邏輯：情緒分低 → 推薦對應冥想

### Phase 3：Week 6–8（App 化 + 習慣提醒）
- PWA（先）或 React Native（後）
- 推播通知（每日冥想提醒）
- RevenueCat 管理 App Store / Google Play 訂閱
- 離線模式：音頻預先下載，無網路也能用

### Phase 4：Month 3+（成長引擎）
- KOL 合作（心理師、瑜珈老師、Podcast 主持人）→ 聯名課程
- 月費型主題課程（焦慮月 / 睡眠月 / 專注月）
- 企業健康方案 → 科技業、金融業 HR 主動接觸
- ATEN 升級付費方案，批量生成完整付費內容庫

## 八、差異化護城河

4. **低定價切入** — NT$990/年 vs 競品 NT$2,100–2,400/年，轉換門檻低

### 10-2. 感恩練習（Gratitude Practice）

**目標時長：5 分鐘｜字數：約 620 字**

**腳本特性**：引導用戶在一天開始前靜下來回想三件小事表達感謝，內容貼近日常生活。因每天想到的事情不同，同一支音頻可長期有效，重複聽不膩。

---
